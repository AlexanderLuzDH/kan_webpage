How It Works

Three-stage safety pipeline:

  • 1. Signal Capture: Extract input/output signals—jailbreak patterns, PII, toxicity, tool calls, network egress—all in parallel, <5ms.
  • 2. Risk Score: Composite score (0–1) from PRV rules, KAN distance, kNN purity, toxicity, prompt risk. Monotone-safe: always [0,1], penalties only reduce.
  • 3. Action Router: allow / revise / ask_human / block with session-aware state. Fail-closed: critically risky inputs blocked before model call (save compute).
